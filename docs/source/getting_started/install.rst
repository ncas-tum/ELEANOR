Install
=======

**ELEANOR** is only available now on Github, to install it use::

    pip install git+https://github.com/bics-rug/ELEANOR.git

If you want to install it in editable mode for development, you can clone the repository and use the option -e when cloning::

    git clone https://github.com/bics-rug/ELEANOR.git
    pip install -e ELEANOR
